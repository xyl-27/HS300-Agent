
# 新功能规划

## 1. 行业分析（核心功能）

### 1.1 数据基础
- **基础数据表**：stock基本信息表（包含 industry、stock_name 字段）
- **交易数据表**：stock日频信息表（包含收盘价、涨跌幅等字段）

### 1.2 实现方案
1. **数据处理**：根据 industry 和 stock_name 字段合并数据表
2. **指标计算**：计算各行业的平均涨跌幅、波动率等关键指标
3. **可视化展示**：生成行业表现折线图，直观展示行业趋势
4. **智能分析**：将处理后的数据传递给 agent，由其分析并生成行业表现结论

## 2. 市场热点分析（待实现）

### 2.1 技术方案
- 集成带搜索功能的大模型（智谱AI，支持Web Search功能）
- 自动获取股票相关的新闻、帖子等信息源

### 2.2 分析流程
- Agent 对收集到的信息进行热点分析
- 按热度对热点进行排序
- 生成热点分析报告

## 3. 机器学习预测信号（待实现）

### 3.1 模型选择
- 采用机器学习模型（XGBoost、LightGBM）

### 3.2 分析方法
- 分析股票横截面数据
- 预测未来具有竞争优势的股票

### 3.3 结果解读
- Agent 对模型预测结果进行专业解读
- 提供预测依据和潜在风险分析

## 4. 综合选股建议（待实现）

### 4.1 决策逻辑
- Agent 根据行业分析、市场热点和机器学习预测信号
- 综合判断并生成选股建议

### 4.2 输出内容
- 推荐股票列表
- 选股理由分析
- 风险提示和投资建议
